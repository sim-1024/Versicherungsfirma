# Versicherungsfirma
Verwaltung der KFZ-Versicherungen, Lebensversicherungen und Patientendaten.

Dieses C++-Programm ermöglicht Benutzern, die relevanten Informationen zu verschiedenen Versicherungen und Patienten zu speichern, zu bearbeiten und anzuzeigen. Das System speichert und verwaltet Details zu Versicherungspolicen wie Versicherungsnummern, Versicherungssummen, Vertragslaufzeiten sowie zu Patienten. Durch die benutzerfreundliche Oberfläche können Versicherungsanbieter und Gesundheitsdienstleister effizient alle notwendigen Daten abrufen und bearbeiten.
